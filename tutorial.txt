TÓPICOS
--> Mysql
    - mysql -u root -p 
    - sudo apt install mysql
    - sudo service mysql start
    - sudo service mysql stop
    - sudo mysqld_safe --skip-grant-tables &
    - mysql -u root
    - CREATE USER 'user'@'localhost' IDENTIFIED BY '123';
    - GRANT ALL PRIVILEGES ON *.* TO 'user'@'localhost';
    - FLUSH PRIVILEGES;
    - EXIT;



--> IP estático
    - sudo ufw allow 8000/tcp
    - para nomear um ip na maquina local
        - sudo nano /etc/hosts 
    - colocar ip e o nome
        - 192.168.0.250  ofmakers
    - sudo nano /etc/dhcpcd.conf
    - static ip_address=192.168.0.250/16

    *DESLIGAR E LIGAR WIFI PARA funcionar*

--> Clonando do github
    - sudo apt-get install git
    - git clone https://github.com/Lecrepus17/ofmakers

--> Laravel (PHP artisan)
    - Acessar a pasta
        - cd ofmakers
        - composer install
        - cp .env.example .env
        - php artisan key:generate
        - php artisan migrate
